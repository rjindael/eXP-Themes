A series of themes in Godot 4.5, meant to resemble some of the themes from Windows XP, including:
- Luna
- Olive (Homestead)
- Embedded
- Metallic
- Royale
- Royale Dark
- Zune

To be used in ongoing game project [nos-pros](https://github.com/alim395/nospros-demo), and thought it would be cool to share.

Feel free to contribute and adapt to your own projects, and would appreciate the credit :)

# Usage

## Method 1 - Release:

Download the [latest release](https://github.com/alim395/eXP-Themes/releases/latest), and copy the folders to the root of you project.

**OR**

## Method 2 - Git Clone:

Clone the repo with the following command:

```
git lfs install --skip-repo
git clone https://github.com/alim395/eXP-Themes.git
```

Then copy the "addons" and "themes" folder to the root of your project.
